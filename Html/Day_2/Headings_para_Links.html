Full Stack Lite Developer

Full Stack Lite Developer wo hota hai jo basic level par frontend aur backend dono ka kaam kar sakta hai. Iska matlab yeh hai ke aap complete software engineer nahi hotay, lekin small websites aur projects bana sakte ho.

Frontend Skills

Frontend ka matlab website ka wo hissa jo user ko screen par nazar aata hai. Full Stack Lite developer ko HTML, basic CSS aur thori si JavaScript aani chahiye.

HTML

HTML website ka structure hota hai. Is se headings, paragraphs, images aur links banaye jate hain.

Basic CSS

CSS website ko style dene ke liye hoti hai jaise colors, fonts aur layout. Full Stack Lite me sirf basic CSS ka knowledge kafi hota hai.

Backend Basics

Backend wo hissa hota hai jo server par kaam karta hai. Full Stack Lite developer ko backend ka sirf basic idea hona chahiye.

Database

Database data ko save karne ke liye use hota hai jaise users ka record ya products ki list.

Freelancing aur Jobs

Full Stack Lite skills ke sath aap freelancing start kar sakte ho aur small jobs hasil kar sakte ho jaise landing pages aur simple websites.
